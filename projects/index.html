Our open contributions to test automation, Neo4j, and elsewhere.

Graph Database IDE plugin

Star

A full-featured graph database plugin for JetBrains IDEs like IntelliJ IDEA andWebStorm. It provides Cypher syntax highlighting, language injection, query execution, autocompletion, and even refactoring.

Selenium Grid Extensions

Star

A collection of Selenium Grid extensions for better UI tests. Some of the features include: downloading files from grid nodes, uploading resources to grid nodes, and running Sikuli tests remotely.

Sterodium RMI

Star

Remote method invocation library for Sikuli Grid extensions. Allows to execute Sikuli code remotely.
